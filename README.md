# Quote Finder: Ideas & Roadmap
Welcome! This repository serves as the central hub for tracking new features, quote requests, and technical improvements for the Quote Finder project.

# 💡 How to Contribute an Idea
We use GitHub Issues to manage our to-do list. Whether you've found a typo, have a killer feature idea, or want to request a specific author, here is how you can help:

Check the existing list: Head over to the Issues Tab to see if your idea is already being discussed.

Open a New Issue: If your idea is unique, click the green "New Issue" button.

Describe it: Give it a clear title (e.g., "Add 'Copy to Clipboard' button") and a brief description of why it would be a great addition.

# 🛠 Current Focus Areas
Quote Expansion: Adding more diverse sources from literature, cinema, and history.

UI/UX: Improving the transition animations between the quote and its source.

Accessibility: Ensuring the site is fully screen-reader friendly.
